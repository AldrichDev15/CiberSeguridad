# 1. Nivel básico

**Objetivo:** Practicar XSS, SQLi, IDOR, LFI, CSRF y fundamentos antes de avanzar en PortSwigger.

### 1.1 DVWA (Damn Vulnerable Web Application)

- Práctica: XSS, SQLi, Command Injection, CSRF, LFI en varios niveles.
    
- Acceso: [https://github.com/digininja/DVWA](https://github.com/digininja/DVWA)
    
### 1.2 bWAPP

- Práctica: +100 vulnerabilidades web, ideal para complementar PortSwigger Apprentice.
    
- Acceso: [https://sourceforge.net/projects/bwapp/](https://sourceforge.net/projects/bwapp/)
    
### 1.3 OWASP Juice Shop

- Práctica: XSS, SQLi, auth bypass, desafíos API básicos.
    
- Acceso: [https://owasp.org/www-project-juice-shop/](https://owasp.org/www-project-juice-shop/)
    
### 1.4 Root-Me (Web – easy challenges)

- Práctica: XSS básico, SQLi básico, IDOR básico.
    
- Acceso: [https://www.root-me.org/en/Challenges/Web-Client](https://www.root-me.org/en/Challenges/Web-Client)
    
- Acceso: [https://www.root-me.org/en/Challenges/Web-Server](https://www.root-me.org/en/Challenges/Web-Server)

# 2. Nivel intermedio

**Objetivo:** Avanzar con APIs, lógica de negocio, SSRF, LFI avanzado, uploads, NoSQLi, autenticación.

### 2.1 WebGoat

- Práctica: IDOR, auth flaws, business logic, API security, mass assignment.
    
- Acceso: [https://owasp.org/www-project-webgoat/](https://owasp.org/www-project-webgoat/)

### 2.3 Root-Me (Web – medium)

- Práctica: SSRF, file upload, template injection, NoSQL injection.
    
- Acceso: [https://www.root-me.org/en/Challenges/Web-Server](https://www.root-me.org/en/Challenges/Web-Server)
    
- Acceso: [https://www.root-me.org/en/Challenges/Web-Client](https://www.root-me.org/en/Challenges/Web-Client)

# 3. Nivel avanzado

**Objetivo:** Chaining, deserialización, SSRF avanzado, race conditions, máquinas completas para pentesting web.

### 3.1 Root-Me (Web – hard)

- Práctica: deserialization, advanced SSRF, race condition, multi-step.
    
- Acceso: [https://www.root-me.org/en/Challenges/Web-Server](https://www.root-me.org/en/Challenges/Web-Server)
    
- Acceso: [https://www.root-me.org/en/Challenges/Web-Client](https://www.root-me.org/en/Challenges/Web-Client)
    
### 3.2 OWASP Broken Web Applications Project

- Práctica: múltiples apps conectadas, ideal para encadenar vulnerabilidades.
    
- Acceso: [https://sourceforge.net/projects/owaspbwa/](https://sourceforge.net/projects/owaspbwa/)
    
### 3.3 VulnHub – Máquinas avanzadas

- Mr Robot, SkyTower, Peppo, Billu:Box.
    
- Acceso: [https://www.vulnhub.com/](https://www.vulnhub.com/)
    
### 3.4 HackTheBox (Medium/Hard web machines)

- Mango, Mirai, OpenAdmin, DevOops, Cascade.
    
- Acceso: [https://www.hackthebox.com/machines](https://www.hackthebox.com/machines)
    
### 3.5 TryHackMe – Avanzado

- Overpass, Attacktive Directory (para chaining), Red Team Web paths.
    
- Acceso: [https://tryhackme.com/](https://tryhackme.com/)
